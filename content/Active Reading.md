# Active Reading

Active reading means engaging deeply with knowledge base content rather than passively consuming it.

## The Three Pillars

### 1. Think About the Material
When reading notes, constantly ask:
- Does this connect to something I already know?
- What questions does this raise?
- How can I apply this knowledge?

### 2. Bring in New Thoughts
Ideas emerge from engagement. Capture:
- Questions that arise
- Contradictions you notice
- Extensions or applications
- Personal associations

### 3. Add New Notes
Immediately create atomic notes from your thoughts:
- Don't trust memory — write it down
- Each thought = one atomic note
- Link to where the thought originated

## Why It Matters

The second brain only grows when you actively contribute. Reading without thinking produces no new knowledge. Every note added is a connection made.
