# Frontier Exploration: Competing Framework Knowledge

> How do you organize knowledge when multiple epistemologically equivalent but mutually incompatible frameworks exist?

## The Problem

Some knowledge domains have **competing frameworks** that are:
- **Internally consistent** — each framework has its own logic
- **Empirically supported** — each has evidence, though different types
- **Mutually incompatible** — they cannot both be true simultaneously
- **Equally valid** — no framework is clearly "wrong"

Examples:
- **Physics**: Copenhagen interpretation vs. Many-worlds vs. Bohemian mechanics (quantum mechanics)
- **Economics**: Keynesian vs. Austrian vs. Monetarist vs. MMT
- **Psychology**: Psychoanalysis vs. Behaviorism vs. Cognitive vs. Humanistic
- **Philosophy**: Various schools (existentialism, utilitarianism, deontology, virtue ethics)
- **Medicine**: Allopathic vs. naturopathic vs. traditional systems
- **Law**: Common law vs. civil law traditions

This is different from:
- **Contradictory sources** — where one is wrong (covered by Seed)
- **Subjective opinions** — where all are valid preferences (covered by Seed)
- **Equivalent approaches** — where context determines best choice (covered by Seed)

## Why Current Rules Are Incomplete

The Seed has rules for:
- Synthesizing contradictory sources (picking one is wrong)
- Presenting subjective perspectives as options
- Context-dependent alternatives

But it lacks guidance for:
1. **Framework neutrality** — How to present frameworks without implicit bias
2. **Evidence hierarchy within frameworks** — Each framework has its own standards
3. **Framework switching costs** — Learning a new framework requires unlearning
4. **Practical integration** — When to use which framework
5. **Framework provenance** — Where each framework originates culturally/historically

## The Challenge

Knowledge bases naturally imply preference through:
- **Presentation order** — First = default = implicit endorsement
- **Link structure** — More links = more important
- **Detail level** — Deeper coverage implies endorsement
- **Language** — Words like "orthodox" vs "alternative" carry judgment

An AI agent building a knowledge base needs explicit guidance to be framework-neutral.

## Proposed Approaches

### Framework-Aware Structure

**Rule:** Create explicit framework hub notes when covering domains with competing frameworks.
**Why:** Without explicit framework hubs, readers (and AIs) cannot distinguish framework-neutral content from framework-specific content.
**Test:** In a domain with competing frameworks, can you identify: (1) framework-neutral core knowledge, (2) framework-specific expansions, (3) explicit framework hub?

### Framework Tagging

**Rule:** Tag notes with explicit framework scope — what framework does this knowledge assume?
**Why:** Knowledge that appears universal may actually be framework-specific. Tagging reveals hidden assumptions.
**Test:** Pick 5 notes in a domain with competing frameworks. Can you identify which framework each assumes?

### Neutral Presentation

**Rule:** Present frameworks in order of historical prominence or audience relevance, not perceived correctness.
**Why:** Presentation order implies endorsement. Alphabetical or historical order is neutral.
**Test:** Can you identify the implied order of endorsement in your framework notes?

### Evidence Standards Within Frameworks

**Rule:** Acknowledge that evidence standards vary by framework — what counts as evidence in one may not in another.
**Why:** A knowledge base that applies only Western scientific standards to all frameworks is not neutral.
**Test:** For each framework, can you identify its evidence standards?

### Practical Guidance

**Rule:** When frameworks have practical implications (medicine, engineering), provide framework-specific guidance for when each applies.
**Why:** Readers need to know not just what each framework says, but when to use which.
**Test:** Can a reader determine which framework applies to their situation?

## Domain-Specific Considerations

### Science
- Competing theories in physics (quantum interpretations)
- Competing models in biology (various evolutionary frameworks)
- Different methodologies (experimental vs. observational)

### Social Sciences
- Competing theories of human behavior
- Different methodological approaches (quantitative vs. qualitative)
- Cultural frameworks for understanding society

### Philosophy
- Competing ethical frameworks
- Different epistemological positions
- Metaphysical alternatives

### Professional Domains
- Different schools of practice (medicine, law, therapy)
- Traditional vs. modern approaches
- Cultural variations in professional standards

## Testable Insights

1. **Framework neutrality is achievable but requires effort** — Without explicit rules, neutral presentation is impossible.

2. **All frameworks have blind spots** — Acknowledging what each framework misses is as important as describing what it includes.

3. **Practical integration is where users need most help** — Theoretical comparison is easy; knowing when to apply which is hard.

4. **Framework choice often reflects values, not just evidence** — Economics and ethics especially.
