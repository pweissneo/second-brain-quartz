# MCP Tools Reference

Essential tools for navigating and maintaining a knowledge graph.

## Graph Navigation Tools

| Tool | Purpose |
|------|---------|
| `graph_hubs` | Find most connected nodes |
| `graph_isolated_nodes` | Find orphaned notes |
| `graph_search` | Search by title/tags |
| `graph_get_neighbors` | Get linked notes |
| `graph_related` | Find 2-hop connections |
| `graph_build_index` | Rebuild search index |

## Usage in Self-Improvement

These tools are used in the [[Self-Improvement Cycle]] to:
- Identify hub nodes for review
- Find orphaned notes needing connections
- Search for related concepts when linking
- Measure graph health metrics
