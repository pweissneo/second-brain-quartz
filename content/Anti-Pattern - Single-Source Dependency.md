# Anti-Pattern: Single-Source Dependency

Depending entirely on one source makes notes fragile and unverifiable.

## Problem

Single-source dependency creates:
- Notes that don't make sense without the original
- No way to verify claims
- Broken knowledge if source disappears

## Signs

- No sources cited in notes
- Notes that are just copied text
- Can't verify any factual claim

## Solution

Always add source provenance and write [[Active Reading|independently]].

- Synthesize in your own words
- Cite multiple sources when possible
- Make notes self-contained

## The Synthesis Test

Can you understand this note without clicking any links? If not, it needs more context.
