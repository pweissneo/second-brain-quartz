# The Knowledge Flywheel

Once started, knowledge builds on knowledge.

## What Is a Flywheel?

A flywheel is hard to start but keeps spinning once moving:
- Initial effort required
- Momentum builds
- Keeps going with less effort

## Knowledge Is a Flywheel

### Starting Is Hard
- First notes feel useless
- Few connections
- No emergence

### Then Momentum Builds
- More notes = more connections
- Connections create insights
- Insights motivate more notes

### Finally, Self-Sustaining
- Adding one note triggers multiple connections
- Insights emerge without forcing
- System improves itself

## The Flywheel Effect

```
Add note → Connect → Emergence → Insight → Motivation → Add note
```

Each turn makes the next easier.

## Racing The Flywheel

Early stages need push:
- Force connections
- Add notes consistently
- Review regularly

Later stages coast:
- Insights emerge
- System improves itself
- Less effort needed
