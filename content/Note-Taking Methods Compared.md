# Note-Taking Methods Compared

Different approaches to capturing and organizing knowledge.

## Zettelkasten (Slip Box)

**Philosophy:** Atomic notes, links, emergence
- One idea per note
- Dense linking
- Emergence through connections
- **Best for:** Research, learning, creative work

---

## PARA (Projects/Areas/Resources/Archives)

**Philosophy:** Action-oriented organization
- Categorize by function, not topic
- Focus on outcomes
- Nested folders + links
- **Best for:** Productivity, work projects

---

## Cornell Method

**Philosophy:** Structured note-taking
- Cue column (questions)
- Notes column (main points)
- Summary section
- **Best for:** Classroom notes, studying

---

## Mapping/Mind Mapping

**Philosophy:** Visual, branching
- Central concept
- Radiating branches
- Visual connections
- **Best for:** Brainstorming, planning

---

## Smart Notes (Digital PKM)

**Philosophy:** Combines approaches
- Atomic notes (Zettelkasten)
- PARA for structure
- Bidirectional linking
- **Best for:** Knowledge workers using digital tools

---

## Which to Choose?

| Need | Method |
|------|--------|
| Research | Zettelkasten |
| Projects | PARA |
| Studying | Cornell |
| Brainstorming | Mind Maps |
| Personal knowledge | Smart Notes |
