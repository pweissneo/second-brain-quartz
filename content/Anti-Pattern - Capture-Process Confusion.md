# Anti-Pattern: Capture-Process Confusion

Trying to fully process notes at capture time breaks the workflow.

## Problem

Confusing capture with processing leads to:
- Notes take hours to create
- No inbox or staging area
- Ideas lost before they can be captured

## Signs

- Every note takes 30+ minutes to create
- No distinction between quick capture and processed notes
- Ideas get lost because processing is too slow

## Solution

Capture quickly, process later. Separate capture from refinement.

- Use an inbox for raw captures
- Process within 48 hours
- Don't try to write perfectly on first pass

## The Two-Phase Workflow

1. **Capture** - Quick save, any format, get the idea down
2. **Process** - Transform into atomic notes with links
