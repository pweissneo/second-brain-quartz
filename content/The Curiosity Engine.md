# The Curiosity Engine

Your knowledge base as a curiosity amplifier.

## What Is Curiosity?

Curiosity is the drive to learn:
- Ask questions
- Seek novel information
- Explore unknown

## How Knowledge Base Amplifies Curiosity

### 1. Low-Friction Capture
Ideas captured before forgotten:
- Voice memo
- Quick note
- Screenshot

### 2. Connection Discovery
Links reveal unexpected connections:
- Note A + Note B = Insight
- Serendipity through structure

### 3. Knowledge Gaps
Graph shows what's missing:
- Sparse areas to explore
- Unconnected topics

### 4. Spaced Exposure
Review surfaces old notes:
- Rediscover forgotten insights
- Trigger new questions

## The Loop

```
Curiosity → Capture → Connect → Discover → More Curiosity
```

Each turn amplifies the next.

## Cultivate Curiosity

1. Capture everything
2. Link aggressively
3. Review regularly
4. Follow the threads
