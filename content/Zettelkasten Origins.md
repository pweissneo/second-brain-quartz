# Zettelkasten Origins

The historical foundation of the linked note-taking method.

## Luhmann's Method

Niklas Luhmann developed the Zettelkasten method:
- German term: "slip box"
- Used as his primary thinking tool
- Produced 70+ books and 400+ articles

## The Key Insight

Luhmann had two filing systems:
1. **Bibliographic notes** - References
2. **Permanent notes** - His own thoughts

The magic: Permanent notes were written in his own words, in sequence, with links.

## The Numbering Myth

Common misconception: Zettelkasten uses sequential IDs (1, 1a, 1b, 1b1...)

Reality: Luhmann used **links**, not IDs. The ID system was a later interpretation.

## The Essential Rules (Luhmann)

1. Write one idea per note
2. Write in your own words
3. Link notes together
4. Add permanent notes sequentially
5. Never delete - only deprecate

## Modern Adaptation

We use wikilinks instead of IDs:
```
[[Concept]] instead of "Note 1a"
```

This is actually closer to Luhmann's original method than the ID system.
