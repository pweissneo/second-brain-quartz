# The Learning Stack

Building a personal knowledge infrastructure.

## What Is a Learning Stack?

A Learning Stack is your integrated system:
- Second brain (notes, connections)
- Capture tools (inbox, clipping)
- Review system (spaced repetition)
- Output system (writing, sharing)

## Components

### 1. Capture Layer
- Quick capture (voice, text, screenshot)
- Inbox for processing
- Sources tracking

### 2. Processing Layer
- Atomic note creation
- Linking and connecting
- Template usage

### 3. Storage Layer
- Second brain (Obsidian/other)
- Graph-based linking
- Search and retrieval

### 4. Review Layer
- Spaced repetition
- Weekly reviews
- Graph maintenance

### 5. Output Layer
- Writing
- Teaching
- Decision-making

## Integration

All layers connect:
- Input flows through capture → process → storage
- Review maintains quality
- Output creates feedback
