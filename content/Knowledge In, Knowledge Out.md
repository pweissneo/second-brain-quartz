# Knowledge in, Knowledge out

The flow from input to output.

## The Pipeline

```
Input → Capture → Process → Connect → Output
```

## Input

Sources of knowledge:
- Books and articles
- Videos and podcasts
- Conversations
- Experience and experiments

## Capture

Quick saving to inbox:
- Don't process, just save
- Add source URL
- Use quick capture tools

## Process

Turn raw input into atomic notes:
- Write in your own words
- Extract key concepts
- Create atomic notes

## Connect

Link to existing knowledge:
- Add 2-3 meaningful links
- Find hidden connections
- Build the graph

## Output

Using your knowledge:
- Writing articles/posts
- Making decisions
- Solving problems
- Teaching others

## The Feedback Loop

Output leads to more input:
- You write about X
- Reader asks question about Y
- Y becomes new input

This creates a virtuous cycle of knowledge growth.
