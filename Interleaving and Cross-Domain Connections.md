# Interleaving and Cross-Domain Connections

Mixing topics to create deeper understanding and unexpected insights.

## The Problem with Block Learning

Most people study one topic at a time:
```
Math → Math → Math → Physics → Physics → Physics
```

This creates isolated knowledge silos.

## Interleaving Solution

Mix topics during learning:
```
Math → Physics → Math → Chemistry → Physics → Math
```

## Benefits

1. **Better retention** - Spacing improves long-term memory
2. **Faster discrimination** - Learn to distinguish concepts
3. **Unexpected connections** - Cross-domain insights emerge
4. **Real-world application** - Life doesn't come in blocks

## How It Works in Your Knowledge Base

### 1. Link Across Domains
When processing notes, actively seek connections to [[Emergence in Knowledge Graphs|different fields]]:
- Economic theory → psychological concept
- Programming pattern → biological system
- Historical event → modern business

### 2. Create Bridge Notes
```
# The Lindy Effect

From: Aging software → Business longevity
Applies to: Knowledge that improves with age
Connection: [[Note Lifecycle Management]] principles
```

### 3. Use Tags for Interleaving
Tag notes with cross-domain indicators:
- `#analogy` - Notes with cross-domain connections
- `#pattern` - Reusable patterns across fields

## Practical Application

| Instead of | Do This |
|------------|----------|
| All physics notes together | Mix with mathematics, chemistry |
| Only history notes | Link to economics, sociology |
| Single-topic folders | Cross-reference widely |
