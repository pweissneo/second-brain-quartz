# Knowledge Management Daily Practice

Practical habits for maintaining a healthy knowledge base.

## Daily Practice

| Time | Action |
|------|--------|
| Morning | Process 3-5 notes from inbox |
| Throughout | Capture new inputs quickly |
| Weekly | Run graph maintenance |

## Tools We Use

- **Obsidian** - Primary note-taking (graph-linked)
- **vault-graph MCP** - Navigate and analyze
- **vault-sources MCP** - Track provenance
- **AI agents** - Can assist with capture, processing, linking, and maintenance
