# Anti-Pattern: Perfectionism

Never publishing a note because it's "not ready" stops all progress.

## Problem

Perfectionism leads to:
- No new notes for weeks
- Notes constantly being revised
- No [[Self-Improvement Cycle|iteration]]

The vault stagnates while you chase an impossible standard.

## Signs

- Notes remain in "draft" for months
- Last note created was weeks ago
- Revisions never complete

## Solution

Write for your future self, not for perfection.

- Use [[Note Lifecycle Management|iterate later]]
- Capture the core idea, refine over time
- Done is better than perfect

## The Iteration Mindset

Notes evolve through stages:
1. Draft - capture the idea
2. Active - refine and link
3. Evergreen - stable, well-connected

Start with draft, improve through iteration.
