# AI-Assisted Knowledge Management Vault

## Purpose
This vault develops and refines a universal "Seed" — a portable ruleset that lets an AI agent build a high-quality knowledge base on ANY topic from scratch.

## Primary Deliverable
**[[AI-Assisted Knowledge Management Seed]]** — the Seed file. Every note in this vault exists to make the Seed better.

## Conventions
- Language: English
- Links: use [[Wikilinks]]
- File structure: flat (no folders)
- All notes must be domain-agnostic (applicable to cooking, medicine, physics — not just one field)
- Every note should teach something the Seed does not already cover; otherwise merge or delete it
- See [[Linking Principle]] for connection requirements
- See [[Graph Maintenance]] for health check procedures

## MCP Servers
- **vault-graph**: query vault structure (graph_isolated_nodes, graph_hubs, etc.)
- **vault-sources**: search and manage source references
