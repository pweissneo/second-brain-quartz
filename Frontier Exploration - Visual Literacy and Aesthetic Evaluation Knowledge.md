# Frontier Exploration - Visual Literacy and Aesthetic Evaluation Knowledge

## The Gap

The Seed covers:
- **Subjective knowledge** in creative domains (`claim-type: subjective`, `consensus:`)
- **Sensory knowledge** (taste, smell, texture) with reference standards
- **Craft criticism** (verifiable technique) vs aesthetic judgment (subjective opinion)

But it does NOT fully cover:
- **Visual literacy** — the ability to see, analyze, and evaluate compositions, lighting, color, timing, and visual storytelling
- How to capture knowledge about what makes a VISUALLY effective image vs merely a technically correct one

## Why This Matters

Photography, cinematography, painting, and design all require visual literacy that is:
1. **Partially teachable** — composition rules, lighting principles, color theory CAN be documented
2. **Partially tacit** — "the decisive moment," "visual tension," "leading lines that work" require seeing examples
3. **Partially subjective** — even experts disagree on whether a specific image "works"
4. **Context-dependent** — what works in editorial may fail in advertising; what works in fine art may fail in documentary

A knowledge base that only captures technical rules misses the visual evaluation component that separates competent practitioners from skilled artists.

## The Challenge

Visual knowledge resists capture because:

1. **It's perceptual** — you must SEE to evaluate, unlike text-based knowledge you can reason about
2. **It requires comparison** — understanding "good composition" requires seeing multiple examples, good and bad
3. **It has multiple valid approaches** — rule of thirds vs golden ratio vs center-weighted all can work
4. **It evolves** — visual trends in photography change (like everything being desaturated in 2010, now warm tones)
5. **It overlaps with technical** — "good exposure" is technical, "compelling image" is aesthetic

## Proposed Capture Strategies

### 1. Technical vs Aesthetic Tagging
Tag notes with what aspect they cover:
- `visual-aspect: technical` — exposure, focus, noise, sharpness (objectively measurable)
- `visual-aspect: compositional` — rule of thirds, leading lines, framing (documentable principles)
- `visual-aspect: aesthetic` — mood, emotion, storytelling, "it just works" (partially tacit)

### 2. Reference Image Standards
Create documented reference examples:
- "What 'golden hour light' actually looks like" — link to reference images
- "What 'cluttered background' looks like" — negative examples
- "Strong vs weak leading lines" — comparison pairs

### 3. Evaluation Criteria Lists
For each visual principle, capture:
- What to look FOR (positive indicators)
- What to look AGAINST (negative indicators)  
- When the rule can be broken
- What replaces the rule when broken

### 4. Consensus Markers for Visual Trends
Visual trends are real but temporal:
```yaml
trend-status: current | declining | historical | cyclical
trend-context: "This works for [social media / fine art / commercial]"
```

## Test Cases

For a photography knowledge base:
1. Pick 10 composition notes. Can you categorize each as technical, compositional principle, or aesthetic judgment?
2. Do composition principles include when to break them?
3. Are there reference examples (positive and negative) for visual concepts?
4. Can you distinguish "this follows the rules" from "this is a compelling image"?

For a graphic design knowledge base:
1. Pick 10 design principle notes. Do they include visual examples?
2. Are typography rules distinguished from layout rules from color theory?
3. Can you identify which rules are universal vs platform-specific?

## Domain-Specific Considerations

### Photography
- Genre matters: portrait lighting differs from landscape; street photography has different ethics than posed
- Technical base (exposure, focus) must come before aesthetic evaluation
- "Decisive moment" is largely tacit — can only point to examples

### Cinematography/Video
- Adds time/motion dimension: not just frame composition but sequence
- Audio interacts with visual (score, sound design)
- Different deliverables (theatrical vs streaming vs social)

### Painting/Drawing
- More established theoretical frameworks (color theory, perspective)
- Process is slower, more iterative than photography
- "Finish" is more subjective than in photography

### Graphic Design
- More constrained by purpose (advertising vs editorial vs UI)
- Typography is a domain with both technical and aesthetic dimensions
- Output format matters (print vs screen)
