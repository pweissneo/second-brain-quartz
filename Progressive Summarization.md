# Progressive Summarization

Progressive Summarization is a note-taking technique developed by **Tiago Forte** (creator of the PARA method) for efficiently processing large amounts of information.

## The Method

Summarize content in **multiple passes**, each more detailed:

| Pass | Format | Purpose |
|------|--------|---------|
| 1 | Bolded text | Capture main points |
| 2 | Bulleted list | Structure emerges |
| 3 | Your words | Personal interpretation |
| 4 | External links | Connect to network |

## Why Progressive?

- First pass is fast (identify what's notable)
- Later passes deepen understanding only when needed
- Balances depth with time efficiency
- Creates layered notes for different contexts

## Connection to Knowledge Management

Progressive Summarization feeds into [[Atomic Note Principle]]:
- Each bolded point can become an atomic note
- Summary layers provide context for linking
- Complements [[Zettelkasten Method]] workflow

## In Practice

1. **Capture** - Save original content (article, transcript, book excerpt)
2. **Distill** - Bold key points (pass 1)
3. **Structure** - Convert to bullets (pass 2)
4. **Synthesize** - Write in your own words (pass 3)
5. **Connect** - Link to related notes (pass 4)

## Comparison with Zettelkasten

Progressive Summarization and Zettelkasten are complementary methodologies:

| Aspect | Progressive Summarization | Zettelkasten |
|--------|---------------------------|---------------|
| Unit | Summary | Atomic note |
| Focus | Compression | Connection |
| Input | Consumable content | Any idea |
| Output | Quick recall | New insights |
| Time | Fast (15 min/chapter) | Slow (1 note/idea) |

**When to use Progressive Summarization:** Processing books or articles, need quick reference summaries, building a reference library

**When to use Zettelkasten:** Doing research or writing, wanting to generate new ideas, building a knowledge network

**Best practice:** Use Progressive Summarization for initial capture, then convert summaries to atomic Zettelkasten notes for linking (Highlight → Bold → Bullet → Extract atomic idea → Write in own words → Link to related notes)
